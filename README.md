# REGRAS FUNCIONAIS
- [ ] O usuário deve poder criar novas transações	
- [ ] O usuário deve poder obter um resumo de sua conta
- [ ] O usuário deve poder listar todas suas transações
- [ ] O usuário deve poder visualizar uma transação

# REGRAS DE NEGÓCIO
- [ ] A transação pode ser:
  - [ ] tipo crédito: somará o valor à fatura da conta
  - [ ] tipo débito: subtrairá o valor ao saldo da conta 
- [ ] O usuário só pode visualizar suas transações
